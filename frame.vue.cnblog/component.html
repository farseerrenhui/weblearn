• 使用script或template标签分离组件

    在定义组件的选项时,data和el选项必须使用函数

  • 可以使用props把父组件的数据传递给子组件

  • prop的绑定类型 👉 1.单向绑定

    父组件数据
    parent_name {{ parent_name }}

    修改父组件数据,子组件数据同时改变

    修改子组件数据,父组件数据不会改变

  • prop的绑定类型 👉 2.双向绑定(Vue2.0不支持了,寻找新的方式)

    父组件数据
    parent_name {{ parent_name }}

    修改父组件数据,子组件数据同时改变

    修改子组件数据,父组件数据不会改变

  • prop的绑定类型 👉 3.单次绑定(Vue2.0不支持了,寻找新的方式)

    父组件数据
    parent_name {{ parent_name }}

    修改父组件数据,子组件数据同时改变

    修改子组件数据,父组件数据不会改变
